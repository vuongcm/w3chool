JavaScript Array push() Method

❮ JavaScript Array Reference

Ví dụ

Thêm một mục mới vào một mảng:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Kiwi");
Tự mình thử »

Thêm ví dụ "Tự mình thử" bên dưới.

Định nghĩa và cách sử dụng

Phương thức push() thêm các mục mới vào cuối một mảng và trả về độ dài mới.

Ghi chú: (Các) mục mới sẽ được thêm vào cuối mảng.

Ghi chú: Phương thức này thay đổi độ dài của mảng.

Mẹo: Để thêm các mục ở đầu một mảng, sử dụng phương thức unshift().

Trình duyệt hỗ trợ

Các số trong bảng chỉ định phiên bản trình duyệt đầu tiên hỗ trợ đầy đủ phương thức.

Phương thức
push() 1.0 5.5 1.0 Có Có

Cú pháp

array.push(item1, item2, ..., itemX)

Giá trị tham số

Tham số Miêu tả
item1, item2, ..., itemX Bắt buộc. The item(s) to add to the array

Technical Details

Return Value: A Number, representing the new length of the array Try it »
JavaScript Version: ECMAScript 1

Ví dụ

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Kiwi", "Lemon", "Pineapple");
